Installation

Add this script tag to your website, just before the closing </body> tag:

<!-- Jarvis Chat Widget --> <script src="https://yourdomain.com/widget-v2.js" data-bot-id="YOUR_BOT_ID" data-position="bottom-right" data-primary-color="#0ea5e9" data-auto-open="false"> </script>

Configuration Options

data-bot-id (required)
Your unique bot identifier. Get this from your dashboard.
data-position (optional)
Widget position: bottom-right, bottom-left, top-right, top-left
Default: bottom-right
data-primary-color (optional)
Primary color for the widget (hex code)
Default: #0ea5e9
data-auto-open (optional)
Automatically open the chat on page load
Default: false
data-api-url (optional)
Custom API endpoint URL
Default: Current domain

Advanced Usage

You can also control the widget programmatically:

// Open the widget window.postMessage({ type: 'JARVIS_OPEN' }, '*'); // Close the widget window.postMessage({ type: 'JARVIS_CLOSE' }, '*'); // Send a message window.postMessage({ type: 'JARVIS_MESSAGE', text: 'Hello, bot!' }, '*');
